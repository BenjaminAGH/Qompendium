### Enunciado
El grupo QuantumUBB requiere implementar un servidor web interno que sea accesible mediante un nombre de dominio para poder promocionar el grupo. Tu tarea es diseñar y configurar una red local en Cisco Packet Tracer que permita a los usuarios acceder al sitio web escribiendo **"quantumubb.cl"** en lugar de utilizar la dirección IP del servidor.

Deberás crear una topología de red compuesta por un **(1)** computador *cliente*, un **(2)** switch y dos servidores: uno que funcionará como **(3)** **servidor DNS** y otro como **(4)** **servidor web**. Todos los dispositivos deben conectarse al switch mediante cables. La red utilizará el esquema de direccionamiento `<192.168.1.0/23>`, donde la PC tendrá la dirección IP `<192.168.1.10>`, el servidor DNS la dirección `<192.168.1.3>` y el servidor web la dirección `<192.168.1.2>`. Todas las configuraciones de IP deben realizarse de forma **estática**, es decir, sin utilizar DHCP.

| Dispositivo    | Dirección IPv4 | Máscara     | Gateway     | DNS        |
|----------------|--------------|---------------|-------------|------------|
| **PC0**        | 192.168.1.10 | 255.255.255.0 | 192.168.1.1 | 192.168.1.3|
| **Server-Web** | 192.168.1.2  | 255.255.255.0 | 192.168.1.1 | NULL       |
| **Server-Dns** | 192.168.1.3  | 255.255.255.0 | 192.168.1.1 | NULL       |
